This project was written in C++ on Linux (Ubuntu 12.04) operating system.

To compile it,
1. Open a terminal
2. Navigate to this directory
3. Run:
	make
4. Run the compiled program:
	./a.out
	
	or
	
	./a.out > log.txt
	if redirection is necessary
	
5. A prompt ">>" is shown on the terminal. Enter a query in
	as many lines you find convenient and press ';' at the end.

6. To stop the execution, enter Ctrl + C.
